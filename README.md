✨ About The Project
A modern e-commerce mobile application built with Flutter and Dart.
This project aims to provide a seamless shopping experience with an intuitive user interface and robust functionality.

🚀 Getting Started
Prerequisites
Flutter SDK

Dart SDK

Android Studio / VS Code

Git

Installation
bash
# Clone the repository
git clone https://github.com/datjpro/ecommerce_flutter.git

# Navigate to the project directory
cd ecommerce_flutter

# Install dependencies
flutter pub get

# Run the app
flutter run
